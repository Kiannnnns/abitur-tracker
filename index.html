Dein Abi. Dein System.

GitHub-Pages-fertiger Abi Tracker: Trage Lernzeit direkt ein, tracke deinen Fortschritt pro Fach, behalte deine Prüfungen im Blick und nutze zusätzlich einen Trading-Tab mit Live-Chart.

Übersicht

Das ist die GitHub-Pages-Version deiner Website. Du brauchst nur die Dateien in ein Repository hochzuladen und GitHub Pages zu aktivieren.

GitHub-ready

Was die Website kann

Tages-Tracking, schneller Zeiteintrag, Lernblöcke, Wochensoll, Prüfungs-Countdown, News und TradingView-Tab.

Was gespeichert wird

Deine Daten werden lokal im Browser gespeichert. Wenn du dasselbe Gerät und denselben Browser nutzt, bleiben deine Einträge erhalten.

Für GitHub Pages

Die Seite ist als statische HTML-Datei gebaut. Kein Build-Schritt, kein npm, kein extra Framework nötig.

Wie du startest

ZIP entpacken, Dateien in GitHub hochladen, dann unter Settings → Pages den Branch main auswählen.

Komplettpaket für GitHub Pages Lokale Assets enthalten Kein Build nötig

Schneller Zeiteintrag

Trage einfach Minuten und Fach ein. Der Fortschritt wird automatisch dem Fach und dem gewählten Tag zugerechnet.

Beispiel: 35 Minuten + Biologie eintragen. Das Dashboard rechnet die Zeit automatisch zum Fachfortschritt dazu.

Optionale Lernblöcke

Mini-System für jeden Tag

Einfach, klar und wiederholbar.

➜ Minuten direkt nach dem Lernen eintragen.
➜ Fach auswählen, nicht alles im Kopf behalten.
➜ Dashboard rechnet den Fortschritt automatisch hoch.
➜ Wochensoll regelmäßig kontrollieren.
➜ Schwierige Fächer zuerst, leichte später.
